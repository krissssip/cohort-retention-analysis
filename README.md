# Cohort Retention Analysis

## 📊 Overview
This project analyzes user retention using cohort analysis. The goal was to evaluate user engagement over time and compare retention between acquisition channels (Promo vs Organic).

## 🛠 Tools
- SQL (PostgreSQL / DBeaver)
- Google Sheets

## 📂 Project Structure
- cohort_analysis_data_cleaning_and_retention.sql – data cleaning and cohort calculation logic

## 🔍 Key Steps
- Cleaned and normalized inconsistent date formats
- Joined user and event data
- Built monthly cohorts based on signup date
- Calculated retention using month offsets
- Segmented users by acquisition channel

## 📈 Key Insights
- Promo users show faster churn
- Organic users demonstrate stronger long-term retention

## 💡 Business Value
- Suggest focusing on organic acquisition for long-term growth
- Improve onboarding for promo users to reduce early churn
